My First Post

这是我的第一篇博客，那就说说我是如何搭建起它的吧。 Hugo 安装Hugo 1.我直接下载了老师传的文件（可查阅Hugo安装指南，找到对应系统的安装操作)，解压后将它放在指定的安装目录，例如：D：\Hugo\bin。接下来要配置环境变量，找到系统变量点编辑然后新建添加该目录。 2.打开命令行即cmd（可直接win+r），输入hugo version，若成功安装会显示： C:\Users\Lenovo>hugo version hugo v0.88.1-5BC54738+extended windows/amd64 BuildDate=2021-09-04T09:39:19Z VendorInfo=gohugoio （我第一次输入hugo version时，并没有显示这些，是因为我在环境系统里没有加入该目录） 3.新建站点 在命令行中输入hugo new site Muzi，创建一个叫Muzi的文件名为Hugo的站点。 创建第一篇文章，放到post目录里。 C:\Muzi>hugo new posts/my-first-post.md C:\Muzi\content\posts\my-first-post.md created 4.选择主题 下载另一个老师发的文件关于主题的，解压，将它放到Muzi里的theme中 5.预览 启动Hugo server在命令行里输入hugo server -D，打开网址即可 然后我就来到了我的第一篇博客这里并编辑了它。
